# KeyHaven

# What is KeyHaven?
**KeyHaven** is a simple MacOS app designed to store your API keys

It is supposed to be a simple and fast app so it is easy to use


# Features:

- It uses the MacOS keychain for keeping your API keys safe
- Alerts you of upcoming expiring API keys
- Folders to organize your APIs form different providers
- You can lock the app for extra security
- **Everything** is OFFLINE and locally stored
